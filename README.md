# Smart Bunk Risk & Academic Outcome Prediction System

A Streamlit-based machine learning dashboard that predicts student academic risk level and final marks, with analytics and personalized recommendations.

## Tech Stack
- Streamlit
- Pandas
- NumPy
- scikit-learn
- Plotly
- Joblib

## Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Features
- Real-time risk prediction (Low/Medium/High)
- Final marks prediction
- Probability gauge and visual dashboards
- Rule-based smart recommendations
- CSV report download
- Model persistence using joblib
